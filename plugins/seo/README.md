# Open Skill Hub SEO

Generative Engine Optimization (GEO) and AI-search SEO skills for AI agents.

GEO optimizes content so AI answer engines — ChatGPT, Perplexity, Google AI Overviews, Gemini,
and Copilot — **cite and surface** it, rather than chasing the traditional SEO goal of ranking
high for clicks.

## Skills

| Skill | Purpose |
| --- | --- |
| [geo-first-seo](skills/geo-first-seo/README.md) | End-to-end GEO workflow: strategy → create or audit content → technical markup → score, so a topic or page becomes more likely to be cited by AI answer engines. |
