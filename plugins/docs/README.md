# Open Skill Hub Docs

Developer documentation skills for AI agents.

These skills help AI agents produce accurate, well-structured documentation for code repositories —
documentation derived from the code itself rather than invented.

## Skills

| Skill | Purpose |
| --- | --- |
| [readme-writer](skills/readme-writer/README.md) | Analyze a repository, detect its type, and draft a clear, structured README from real commands and examples found in the code — or audit and improve an existing README. Asks only for intent it cannot infer, and never fabricates commands, APIs, or features. |
